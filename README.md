# Инструкция по запуску проекта

Для запуска проекта выполните следующие шаги в указанном порядке:

1. **Запустите СУБД PostgreSQL**  
   Самый удобный способ - использовать Docker. Для этого есть подготовленный файл `docker-compose.yml`.  
   ```
   docker compose up -d
   ```

2. **Настройте переменные окружения**  
   Создайте файл .env в корневой директории проекта и заполните его на основе примера из .env.example.

3. **Установите зависимости**  
   Рекомендуется использовать виртуальное окружение (venv или poetry).  
   ```
   pip install -r requirements.txt
   ```

4. **Запустите бота**  
   Убедитесь, что вы находитесь в корневой директории проекта, и выполните:  
   ```
   python -m app.main
   ```
   
## Описание переменных окружения

| Переменная      | Описание                                                                |
|-----------------|-------------------------------------------------------------------------|
| `BOT_TOKEN`     | Токен бота, необходимый для связывания с вашим ботом через API Telegram |
| `DB_NAME`       | Название базы данных, созданной в СУБД                                  |
| `DB_USER`       | Имя пользователя СУБД Postgres (по умолчанию - `postgres`)              |
| `DB_PASSWORD`   | Пароль от пользователя Postgres                                         |
| `DB_HOST`       | Хост, на котором поднята СУБД                                           |
| `DB_PORT`       | Порт, на котором СУБД слушает входящие запросы                          |
| `DEBUG`         | Опциональный параметр, отвечает за уровень вывода логов                 |
